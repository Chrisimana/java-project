## 🚀 Cara Memulai

### Prasyarat
- **Java 8+** atau versi yang lebih baru
- **JDK (Java Development Kit)** - untuk compile dan menjalankan
- **IDE Java** seperti IntelliJ IDEA, Eclipse, atau NetBeans (opsional)

### Instalasi & Kompilasi

1. **Clone atau download repository ini**
   ```bash
   git clone <repository-url>
   cd java-project
   ```

2. **Navigasi ke folder proyek yang ingin dijalankan**
   ```bash
   cd projects/....
   ```

### Menjalankan Proyek

Setiap proyek memiliki file `Main.java` sebagai entry point:

```bash
# Compile
javac src/**/*.java

# Jalankan
java -cp src Main
```
